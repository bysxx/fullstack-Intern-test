## 과제 설명

프리미엄 패션 브랜드의 상품 리스트 페이지

### 필수 요소

1. **히어로 섹션**
   - 헤드라인
   - 서브카피
   - CTA 버튼

2. **상품 카드 그리드**
   - 최소 4개 상품
   - 카드 구성: 이미지, 브랜드명, 상품명, 가격

3. **product.json 활용**
   - `product.json` 파일 활용
   - 서버에서 JSON 파일을 읽어서 렌더링

### 옵션

- 호버 인터랙션
- 반응형 레이아웃
- 찜하기 버튼
- 스크롤 애니메이션
- 정렬 기능
